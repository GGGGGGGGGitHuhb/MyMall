## 通用规范
### 请求格式
- JSON
- Base URL: /api/v1
### 响应格式（统一返回结构）
```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

| 字段 | 类型 | 含义 |
| --- | --- | --- |
| code | int | 0成功，非0失败 |
| message | string | 错误信息或成功信息 |
| data | object | 返回的数据 |

错误码规范：

| 代码 | 含义 |
| --- | --- |
| 0 | 成功 |
| 1001 | 用户名已存在 |
| 1002 | 密码错误 |
| 1003 | 未登录 |
| 1004 | 无权限 |
| 2001 | 商品存在 |
| 2002 | 库存不足 |
| 3001 | 购物车为空 |
| 4001 | 地址不存在 |
| 5001 | 订单不存在 |
